# Core Engineering Invariants

This document defines the unbreakable rules of the Frugal Fortress architecture. Any Pull Request violating these invariants will be rejected by the CI/CD pipeline or code review.

## 1. State & Resource Invariants

1. **Zero Leaked Sessions:** Every database session MUST be acquired via an `async with` context manager (`sessionmanager.session()`). No exceptions.
2. **Non-Blocking Event Loop:** CPU-bound operations (e.g., Token counting, ONNX Reranking, Cryptography) MUST be offloaded using `asyncio.to_thread()` or the `CpuBoundExecutor`. The FastAPI event loop must never block.
3. **Idempotent Side Effects:** Background jobs (`Arq`) that perform external side effects (LLM calls, DB writes) MUST be idempotent. Use `RedisIdempotencyGuard` keyed by the input hash (e.g., `SHA-256` of the audio/PDF bytes).

## 2. FinOps & Telemetry Invariants

1. **No Silent LLM Calls:** Every LLM request (success or failure) MUST be logged via the `FinOpsCallback`. It must record `model_name`, `prompt_tokens`, `completion_tokens`, and `cost_usd`.
2. **Strict Tenant Isolation:** Every database query, feedback log, and cache key MUST include the `tenant_id`. Cross-tenant data leakage is mathematically impossible if repositories enforce `WHERE tenant_id = :tenant_id`.
3. **Trace Context Propagation:** The `X-Request-ID` generated by the API Gateway MUST be propagated to all background workers and injected into every `structlog` entry and OpenTelemetry span.

## 3. Security Invariants

1. **Fail-Secure DLP:** If the local DLP Proxy (Llama-3.2-3B) crashes or times out, the system MUST throw a `SecurityPolicyViolation` (HTTP 403). It must never fail open and send unredacted text to external APIs.
2. **Application-Level Encryption (ALE):** Sensitive user data (Flashcards, Receipts, Glossary Rules) MUST be encrypted via `Fernet` (AES-128-CBC) before being passed to SQLAlchemy.
3. **No Secrets in Logs:** The `structlog_secret_redactor` MUST sanitize all log outputs. API keys and JWTs must never be written to `stdout` or Loki.
