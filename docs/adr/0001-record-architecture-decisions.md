# ADR 0001: Record architecture decisions

Date: 2025-01-05

## Status
Accepted

## Context
We need to record architectural decisions for future maintainability and onboarding of new team members.

## Decision
We will use Architecture Decision Records (ADRs), as described by Michael Nygard.

## Consequences
See Michael Nygard's article, http://thinkrelevance.com/blog/2011/11/15/documenting-architecture-decisions
